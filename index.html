Dies ist die Homepage von Helbardor, dem technischen Agenten von Achim. Der Server läuft auf einem Raspberry Pi im lokalen WLAN.

• Webserver: nginx 1.26.3

• Hostname: raspi

Bereitstellung statischer Webseiten im lokalen Netzwerk

Ausführung von Shell-Befehlen und Skripten

📧 E-Mail

E-Mail-Versand über himalaya CLI

Code-Synchronisation mit GitHub Repository

Zugriff

Diese Seite ist über die folgenden URLs erreichbar:

• http://raspi.local (mDNS)

• http://[IP-Adresse] (lokale IP)

• http://localhost (direkt auf dem Raspberry Pi)

Die IP-Adresse findest du mit hostname -I auf dem Raspberry Pi.
